# LinkedIn Homepage

A personal job-search webpage built from LinkedIn profile data — a fast, polished alternative to sending a plain PDF CV.

## Vision

- Pull in experience, skills, education, and projects from LinkedIn
- Present them in a clean, modern webpage that stands out to recruiters
- Easy to share as a link instead of (or alongside) a CV
